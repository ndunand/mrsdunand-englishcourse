# English with Mrs Dunand

Static website for upper-secondary English classes.

## Edit class passwords

Open [script.js](/Users/ndunand/Dropbox/Documents/Lizzie/tmpwebsite202606/script.js) and replace the placeholder passwords:

- `change-me-1m02`
- `change-me-2ec1`
- `change-me-3m03`
- `change-me-3m14`

## Add resources

Open the class page you want to update:

- [1m02.html](/Users/ndunand/Dropbox/Documents/Lizzie/tmpwebsite202606/1m02.html)
- [2ec1.html](/Users/ndunand/Dropbox/Documents/Lizzie/tmpwebsite202606/2ec1.html)
- [3m03.html](/Users/ndunand/Dropbox/Documents/Lizzie/tmpwebsite202606/3m03.html)
- [3m14.html](/Users/ndunand/Dropbox/Documents/Lizzie/tmpwebsite202606/3m14.html)

Replace any placeholder `href="#"` item with a real link to:

- PDFs
- PowerPoints
- videos
- Quizlet sets
- glossaries
- revision materials

Example:

```html
<li><a href="files/essay-planning-guide.pdf">Essay planning guide</a></li>
```
